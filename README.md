# Koji Japanese Coaching Version 3.0 — Brand Edition

松下 剛治（Koji Matsushita）の日本語・就職面接コーチングサイトのブランド完成版です。

## 主な更新

- 人物と価値提案を中心にした新しいトップページ
- プロ品質のプロフィールセクション
- 12言語すべてに同じブランド構造を適用
- プロフィール写真とサービス写真の比率・色調を統一
- 外部ライブラリ不要のSVGアイコン
- 色、余白、角丸、影、文字階層を再設計
- モバイルメニューと固定問い合わせバーを改善
- レッスン診断のアクセシビリティと進捗表示を改善
- 新しいOGP画像とfavicon
- 紹介動画の実装枠と一括有効化ツール

## 公開方法

このフォルダの中身をGitHubリポジトリのルートへ上書きし、`main`ブランチへ反映してください。現在のNetlify設定では自動公開されます。
