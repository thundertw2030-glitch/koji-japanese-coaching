# 紹介動画の追加方法

現在、紹介動画セクションは12言語すべてに実装済みですが、未完成表示を出さないため非表示になっています。動画を用意した後は、付属スクリプトで一括反映できます。

## 推奨動画仕様

- 長さ：30〜60秒
- ファイル形式：MP4（H.264映像＋AAC音声）
- 画面：横向き16:9
- 解像度：1920×1080または1280×720
- 容量の目安：20MB以下
- 内容：氏名、支援対象、強み、レッスン方法、相談への一言
- 字幕：可能であれば英語字幕付き。将来は各言語字幕を追加できます。

## 自動反映

プロジェクトのルートで次を実行します。

```bash
python tools/enable_intro_video.py /動画の保存場所/koji-introduction.mp4
```

この処理により、動画が `assets/video/koji-introduction.mp4` にコピーされ、12言語のトップページにある動画セクションが有効になります。

## 公開前確認

1. パソコンとスマートフォンで再生できる
2. 音声が明瞭である
3. 字幕が画面端に近すぎない
4. 動画停止中の表示画像が自然である
5. Netlify公開後も動画URLが404にならない
